We provide the download packages for easy mirroring using rsync. You can mirror the contents at any time, using these command lines: 

  rsync -t --delete rsync://cool.haxx.se/curldownload

We recommend syncing at least daily so that updates take effect swiftly enough.

If you decide to run a download mirror for us (or if you decide to no longer run one), please tell us at curl-web at haxx.se so that we can add it to or remove it from the latest.pl script on the web site. #include "_footer.html"
